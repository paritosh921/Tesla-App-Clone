## Techs :rocket:

- [x] [React Native](https://reactnative.dev/);
- [x] [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript);
- [x] [Framer Motion](https://www.framer.com/motion/);
- [x] [Styled Components](https://styled-components.com/).

<br>

## Installation :wrench:

First you need to clone the project using `git clone https://github.com/paritosh921/Tesla-App-Clone.git`.

You can install the application using `npm install` or `yarn install` on the root dir.

<br>

## Start :on:

To start the application interface just run `npm start` or `yarn start` on the root dir.
